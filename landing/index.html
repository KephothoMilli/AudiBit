How to run a Full Audit

Using the AudiBit extension is seamless. Our agentic dashboard lives right in your browser tools.

1
Open the DevTools Panel

2
Dispatch an Agent
Click on the UI, UX, DOM, or Security agent from the dashboard. The extension will automatically verify your USDC balance and settle the nanopayment on Arc L1.
3
Review Findings
The agent will analyze the live page structure and return a categorized list of issues, along with actionable code snippets to fix them.
